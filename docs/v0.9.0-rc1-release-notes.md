# v0.9.0-rc1 Release Notes

这是 PaperResearch Agent 的真实验收候选版本，不是 v1.0.0。

本候选已经在 Docker Desktop Linux Engine 上完成无缓存构建和五服务启动，验证了真实 PDF 从上传、解析、切分、Hash Embedding、Qdrant 入库到混合检索、Rerank、抽取式问答和页码引用的闭环。PostgreSQL 与 Qdrant 数据在容器重启后保留。

RC 还包含固定主题的 3 篇真实外部论文链路、30 篇中等规模负载、100 次检索、100 次问答、3 次 Deep Research、六组消融，以及主机 Tesseract 对文本/混合/全扫描 PDF 的 OCR 验收。

请特别注意：当前 LLM、Embedding 和 Reranker 分别是未配置、Hash 基线和词法基线；50 条评测数据没有任何人工批准；Semantic Scholar 匿名调用本轮返回 429；Docker 镜像内 OCR、Redis 应用接入和持久 LangGraph checkpoint 尚未完成。因此本版本只适合 RC 演示、工程复验与人工评审。
