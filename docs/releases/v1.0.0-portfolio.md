# PaperResearch v1.0.0-portfolio

## Highlights

- PDF/OCR ingestion with structured blocks and chunks.
- Hybrid retrieval over Jina embeddings, lexical signals, Qdrant, and RRF.
- DeepSeek structured Claim QA.
- Claim-level citation identifiers and deterministic context/page/block validation.
- LangGraph Deep Research with trace and checkpoint persistence.
- PostgreSQL checkpoint recovery.
- Redis, Qdrant, and PostgreSQL integration.
- Docker Compose deployment.

## Verified evidence

- 50/50 Full QA engineering completion.
- 30-minute Portfolio stability test.
- Docker OCR roundtrip for text, mixed, and scanned PDFs.
- PostgreSQL restore and Qdrant snapshot restore.
- 599 automated tests.

## Limitations

- The 50-record dataset is an internal development evaluation set, not an independent blind benchmark.
- Semantic claim support is not formally validated at large scale.
- Retrieval generalization evidence is diagnostic-only.
- The 30-minute stability result is a bounded engineering check, not a long-term production endurance proof.
